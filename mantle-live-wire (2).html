The desk that reads the chain and writes the rundown

Live Wire is an onchain news desk for Mantle. Every headline on this page was written from data it pulled live, not from a press release. It reads chain TVL, every protocol's Mantle position and its daily move, stablecoin supply, and token prices, then ranks what moved and reports it. No model guessing, no API key, no server. If a feed is quiet, that line just goes dark instead of making something up.

ERC-8004 identity

An autonomous desk should publish under a real onchain identity, so every rundown is attributable to the agent that wrote it.

Agent skills

Each data pull, chain TVL, movers, stablecoins, prices, is a composable skill the desk runs on every refresh.
